### `string interpolation`

<br>

**without format**

```python
variable = 'World'
output = f'Hello {variable}!'

# Hello World

```

<br>

**using format**

```python
template1 = 'Hello {}!'
output1 = template1.format('World')

template2 = 'Hello {variable}!'
output2 = template2.format(variable='World')

template3 = 'Hello {0}!'
output3 = template3.format('World')

# Hello World

```
